# Principles of Programming II - Laboratory Works

This repository contains a collection of laboratory assignments and practical works completed during the **Principles of Programming II** course at the **Kazakh-British Technical University (KBTU)**.

## Project Overview
The focus of this course was to deepen the understanding of programming logic, problem-solving, and working with data structures using Python. The tasks are structured into sequential laboratory works located in the corresponding folders.

## Technologies & Tools Used
* **Programming Language:** Python 
* **Environment:** Jupyter Notebook (`.ipynb` files)
* **Core Concepts:** Data analysis, algorithms, and practical programming workflows.

##  Repository Structure
* `LAB1.ipynb` to `LAB10` - Individual assignments covering specific topic requirements, algorithm implementations, and data manipulation tasks assigned throughout the semester.
